# Blogging Website  
Content & Category-Based Blog Platform

A responsive blogging web application built using **React** that displays categorized blog content in a clean and organized layout. The platform includes featured and trending sections, making it easy for users to discover and read articles across different categories.

**Live Demo:** https://saisurya01.github.io/blogging_website/

---

## Overview

This Blogging Website is a front-end React application designed to showcase blog content by category. It focuses on readability, structured navigation, and reusable components to ensure scalability and maintainability.

The project demonstrates how to build a content-driven platform using modern React practices and responsive UI design.

---

## Features

- Category-based blog listing (Technology, Fitness, etc.)
- Featured and trending blog sections
- Clean and responsive layout
- Reusable React components
- Easy navigation between categories
- Optimized for desktop and mobile devices

---

## Technologies Used

- React.js
- React Bootstrap
- JavaScript (ES6+)
- HTML5
- CSS3
